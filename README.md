# Trabalho-Final-N2-Banco-de-Dados

Este projeto foi desenvolvido como Trabalho Final da disciplina Laboratório de Banco de Dados, com o objetivo de criar uma aplicação completa que integre um banco de dados relacional (MySQL) e um banco NoSQL, além de frontend e backend funcionais. O sistema implementa autenticação de usuários, operações CRUD, controle de acesso, uso de triggers, views, procedures, functions e índices devidamente justificados, demonstrando o domínio dos principais recursos de um SGBD e a aplicação prática de conceitos de banco de dados relacionais e não relacionais.
